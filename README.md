# Customer_Churn_Prediction
This project uses machine learning to predict customer churn based on historical data. Data was cleaned and analysed using Pandas, followed by exploratory analysis to identify trends. A classification model was built and evaluated using Scikit-learn to generate insights and support data-driven decision-making.

# Customer Churn Prediction

This project predicts customer churn using machine learning.

- Scikit-learn
